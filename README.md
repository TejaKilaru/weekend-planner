# weekend-planner

A WebApp to post trips and signup for trips.
 
## Requirements
Java 8, Maven, MySql
  
## Run

* Git Clone 
* run sql file (Download .sql from wiki)
* Make Changes in application.properties
* mvn clean install 
* Place the created war file in tomcat's directory
